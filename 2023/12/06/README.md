자료형
	기본 타입 : 8개
	논리 타입
	- boolean 	<- 참/거짓 = 1비트(1바이트)의 크기를 가진다
	
	문자타입
	- char 		<- 문자 (글자 하나) = 2바이트의 크기를 가진다
	
	정수타입
	- byte 		<- -128 ~ 127 = 1바이트의 크기를 가진다
	- short		<- -32768 ~ 32767 = 2바이트의 크기를 가진다
	- int		<- 4바이트의 크기를 가진다 
	- long		<- 8바이트의 크기를 가진다
	
	실수타입
	- float		<- -3.4*10^28 ~ 3.4*10^28 = 4바이트의 크기를 가진다
	- double	<- -1.7*10^308 ~ 1.7*10^308 = 8바이트의 크기를 가진다
	
	레퍼런스 타입 : 3개
	- Array
	- class
	- interface

	- String 클래스(<- 레퍼런스 타입)
		여러개의 문자를 합친 것
